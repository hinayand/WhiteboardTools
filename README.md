# WhiteboardTools
这是一个使用Python和flet库编写的给智能白板使用的工具箱

这个工具箱目前只有一些基础的功能，所以可能会看起来比较简陋

## 我们已经实现了的功能
- 快速系统电源操作
- 小工具
  - 时钟
  - 抽学号
  - 计时器
- 小工具的设置
